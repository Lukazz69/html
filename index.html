Para adicionar audio

Para tocar audio é basicamente simples, voce irá usar a tag <AUDIO>, com os parâmetros de CONTROLS para aparecer o controlador da mídia, e também o parâmetro autoplay.

Outra maneira para adicionar audio no seu site é usando o SOURCE, analise o codigo a seguir.

        
            audio preload="metadata" autoplay controls>
               source src="midia/SONG NO COPYRIGHT - MUSICA SEM DIREITOS AUTORAIS _ Lofi hip hop beat (online-audio-converter.com).mp3" type="audio/mpeg"
               source src="midia/SONG NO COPYRIGHT - MUSICA SEM DIREITOS AUTORAIS _ Lofi hip hop beat (online-audio-converter.com).ogg" type="audio/ogg"
               source src="midia/SONG NO COPYRIGHT - MUSICA SEM DIREITOS AUTORAIS _ Lofi hip hop beat (online-audio-converter.com).wav" type="audio/wav"
            audio

Assim você consegue uma gama de formatos e também de hierarquia de audios.

Seletores em CSS

Temos alguns seletores como CLASS, ID, VALUE e seletores universais. São eles respectivamentes, CLASS (.class) para chamar a função, ID (#id), p[value=""], e o Universal é apenas *{}, O Class pode ser usado em vários elementos, já o ID apenas em todo o código HTML, exemplo ID= "MAIN", so pode tem um MAIN em todo o código. o p[value=""] é para marcar Paragrafos, Listas ou titulos com algum valor especifico. O universal é para todos.

Fontes e Cores.

A muitas maneiras de representar cores no CSS, a mais usada são as cores em HEXADECIMAL, sites que voce pode encontrar alguns exemplos sao o adobe e o colorhunt. Já as fontes você consegue encontrar no próprio visual code (VSCode) como também no GoogleFonts. As fontes existem por caterias que você pode entrar como serifadas, sem serifas, monoespaçadas, fantasia, feitas a mão entre outras... Enfim com o código já copiado basta apenas colar na aba <head>
